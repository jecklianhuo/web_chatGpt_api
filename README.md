# chatGPT H5 backend
很简单的不足200行代码实现多线程和异步应答

## 依赖
基于 https://github.com/yi-ge/PyChatGPT
修改而来，原本这个项目是利用登录方式获取token来验证，但是chatgpt做出了机器人检测，所以已经无法登录，于是我改进为使用openai库来调用API的方式
